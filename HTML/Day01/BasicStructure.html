Head Tag

This is the setup section of your webpage.
It doesnâ€™t show up on the page itself but gives important information and instructions to the browser.
Things you usually put inside head tag:
Title of the page (what shows in the browser tab).
CSS links (for styling).
JavaScript links (for functionality).
Meta tags (information like character set, description, keywords, mobile responsiveness).

Body Tag

This is the visible section of your webpage.
Whatever you want the user to actually see (text, images, buttons, videos, forms) goes inside the body.
The browser displays everything insidebody directly to the screen.
